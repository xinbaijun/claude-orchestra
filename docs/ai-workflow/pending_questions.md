# Pending Questions — 待用户确认问题集中区

> **用途**：所有需要用户拍板才能继续的问题，**不要只在对话里问完就算**，必须 append 到这里。这样你打开一个文件就能看完所有待办，不用挨个翻对话。
>
> **规则**：
> - 任何 Claude 对话遇到"需要用户确认才能继续"的问题，必须在这里登记
> - 已解决的问题：勾掉 `[ ]` 改成 `[x]`，并在 **解决记录** 段追加日期 + 结论
> - 已解决条目 7 天后可清理
> - 同一问题不要重复登记，先 grep 看一眼

---

## 模板

```md
- [ ] YYYY-MM-DD｜模块｜问题描述（一句话）
  - **背景：** 为什么会有这个问题
  - **影响：** 不回答会卡住什么 / 错答的代价
  - **建议选项：**
    - A. ……（推荐理由）
    - B. ……
    - C. ……
  - **关联任务：** resume_here.md 里哪条 / 哪个 PR / 哪个 commit
```
